## Troubleshooting

### If Prowlarr search returns 0 results:
1. Check indexers are enabled in Prowlarr UI
2. Try searching manually in Prowlarr UI to verify it works
3. Verify API key is correct

### If Chillproxy returns error:
1. Make sure Chillstreams is running on port 3000
2. Make sure Chillproxy is running on port 8080
3. Check that the base64 config is valid

### If pool key error:
1. Verify user exists: Check Chillstreams database
2. Verify pool key exists: Check pool tables
3. Verify user is assigned to pool key
